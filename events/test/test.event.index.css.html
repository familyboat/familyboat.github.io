又一年的七夕到了。回想起这几年的七夕，只有17年的七夕是最难忘的。 我记得那年的七夕前夕，我们一起商量着要回学校一起过，还计划了去哪玩。可惜的是，那时候还不太爱拍照，要是能有些照片留下就好了。 我们俩这些年，也不知道是怎样过来的，留下的回忆实在是太少。现在是既不能好好的把握未来，也没法甜美的回忆过去。因为总是一会就回忆空了。 今天又是到了这个美好的节日，我想做点什么，写个网页记录一下吧。记录一下我的心情，记录一下我想做的事情。 最后，附上一张想念的照片及人。
